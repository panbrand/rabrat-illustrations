# Composition Patterns

Use one pattern per image.

## Conflict pull

Use for competing definitions, stakeholder tension, or meaning conflict. Draw an object being pulled by ropes. RaBrat bites, cuts, ties, or secretly redirects one rope.

## Sorting machine

Use for research, filtering, coding, curation, or content operations. Draw a hand-cranked machine. RaBrat turns the crank while scraps enter and labeled outputs leave.

## Pressure press

Use for extracting a core value or insight. Draw a low-tech screw press squeezing plates inward. RaBrat operates the crank and grins.

## Wobbly support

Use for models that need maintenance or management. Draw a hanging mobile, leaning shelf, or crooked triangle. RaBrat props it up with a jack stand or pump.

## Before / after

Use for messy-to-clear transformation. RaBrat performs the decisive action between two states.

## Mini comic

Use for behavior change or failure-to-success arcs. Keep to 2-3 panels. Do not become a sticker pack.
